# Buscando en las Sombras

Un juego interactivo desarrollado con HTML, CSS y JavaScript.

## Descripción

Juego tipo punto y click donde explorarás diferentes espacios buscando objetos y resolviendo misterios en las sombras.

## Archivos

- `index.html` - Página principal
- `styles.css` - Estilos del juego
- `scripts.js` - Lógica del juego
- `objetos/` - Carpeta con imágenes de objetos del juego

## Cómo jugar

Abre `index.html` en tu navegador para comenzar a jugar.
